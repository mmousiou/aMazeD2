Information

aMazeD is an educational game, developed using Blockly library and Blockly Games.

Game's code is open and free to use on GitHub .

Functionality

Game consists of two basic categories.

  • Levels 1-6 are the first category based on maze game. Player must move the character through the selected path with the use of blocks.
  • Levels 7-10 are the second category of the games based on Turtle game. Player must use the blocks in order to draw the shape.

To execute and correct the answers the player has the following buttons:

Button "Play"

With this button you can execute the code and see the results of the animation. If the code is incorrect you have to use "reset" button.

Button "Reset"

With this button you don't execute the code and your character returns to the starting point.

Button "Submit"

With this button you can execute the code, check the result and save it. Whether the result is right or wrong you will be redirected to the next level.

Blocks

  • Move forward
    Moves the player forward one step.

  • Turn left/right
    Turns the player left or right by 90 degrees.
  • If path ahead/to the left/to the right
    If there is path in the specified direction, then do some actions. For example if the do statement has the block "move forward" the the character will move on step forward every time there is a path (ahead/to the left/to the right) depending on the specified direction.
  • If path ahead/to the left/to the right do..(statement)..else..(else statement)
    If there is a path in the specified direction, then do the first block of actions. Otherwise, do the second block of actions.
  • Repeat n-times
    Do some statement several times.
  • repeat until...finish
    Repeat enlosed actions until finish point is reached.
  • Move forward/backward by k-pixel
    Moves the brush forward or backward by the specified pixels.
  • Turn right by/left by μ degrees
    Turns the brush left or right by the specified number of degrees.
  • Set color to...
    Changes the color of the brush.

Results

When the 10th level is completed the game redirects to the results page with the score and the data of all levels. Users can see their results after entering name,surname and a valid e-mail account.

× User data is used only locally for statistical reasons. With the option "New Game" all data is deleted permanently.
